# Security Policy

## Reporting a Vulnerability

Do not open a public GitHub issue for security vulnerabilities.

Report to: **security@zhilicon.ai**

See the [SDK Examples SECURITY.md](https://github.com/zhilicon-ai/zhilicon-sdk-examples/blob/main/SECURITY.md) for our full policy.

## Scope for This Repository

- Benchmark scripts that could be used to extract non-public hardware information
- Harness code with command injection or path traversal vulnerabilities
- Result files that accidentally contain confidential performance data not cleared for publication
